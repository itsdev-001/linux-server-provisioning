# Automated Linux Server Provisioning

This project automates Linux server setup on AWS EC2 using Bash scripting.

## Features

- System update & upgrade
- User creation with sudo access
- Nginx installation
- UFW firewall configuration
- Logging system
- Error handling

## Usage

1. Configure username in config/settings.conf
2. Run:

sudo ./server-setup.sh

## Tech Used

- Bash
- AWS EC2
- Nginx
- UFW
- Git & GitHub
